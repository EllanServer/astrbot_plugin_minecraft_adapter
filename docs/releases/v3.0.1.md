# MineSentinel v3.0.1

MineSentinel 3.0.1 优化事件处置单的阅读体验，让报告更像一份直接、自然的管理说明。

## 报告文案优化

- 将事件详情页中的“AI 给出的详细步骤”和“给维护人员的详细步骤”统一改为“接下来怎么做”。
- 不再在标题中强调建议由 AI 或规则生成，读者可以直接关注需要执行的操作。
- AI 方案来源、安全校验和证据范围说明仍会保留在步骤下方，不影响报告的可追溯性。

## 兼容性

- AstrBot `>= 4.10.4`
- Python 3.10–3.14
- Windows x86_64
- Linux x86_64 / aarch64
- macOS arm64

## 升级

已安装 MineSentinel 的用户可直接使用 README 中的升级提示词更新到 `v3.0.1`。升级不会修改现有日志源、事件状态或报告配置。
